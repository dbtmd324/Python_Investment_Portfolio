# 파이썬 투자 포트폴리오 만들기

## 개요: 퀀트란 무엇인가?
--
퀀트(Quant): 모형을 기반으로 금융 상품의 가격을 산정하거나, 이를 바탕으로 투자하는 사람
- 퀀트 투자는 규칙에 가빈하기에 객관성이 높음
- 데이터와 규칙만 있다면 한 번에 수백, 수천 종목에 대한 분석도 가능
- 규칙에 기반하므로 감정에 휘둘릴 위험이 낮음
- 동일한 규칙을 누구든지 반복할 수 있음
- 규칙을 철저하게 지킬 경우 위험 통제 능력이 높음
